# Architecture Decision Records

状态：active
最后更新：2026-06-26
职责边界：保存 LifeMesh 的重要产品、架构、数据和治理决策。

## 什么时候写 ADR

- 选择或放弃关键技术栈。
- 改变数据层分层。
- 改变权限、审计、撤销策略。
- 新增高敏感数据接入。
- 新增 Agent 自动执行能力。
- 改变路线图阶段顺序。

## 编号规则

使用 `ADR-0001-title.md` 格式，编号递增。
